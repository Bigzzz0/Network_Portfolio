## 🚀 Final Project

### 🪐 Interplanetary Network Simulation

> จำลองระบบเครือข่ายสำหรับการสื่อสารระหว่างดาวเคราะห์ที่มีค่า **delay สูง** และ **link ไม่เสถียร**  
> ประยุกต์ใช้ Delay-Tolerant Networking (DTN) ร่วมกับ Store-and-Forward

[![Repo](https://img.shields.io/badge/Repository-interplanetary--network-181717?style=for-the-badge&logo=github)](https://github.com/Bigzzz0/interplanetary-network)

![Interplanetary Network Topology](Image/Interplanetary_Network.png)

**Key Features:**

| Feature | Description |
|---------|-------------|
| 🪐 Delay-Tolerant Networking (DTN) | รองรับ delay ระดับนาทีถึงชั่วโมง |
| 📡 Store-and-Forward | เก็บข้อมูลไว้รอจนกว่า link จะพร้อม |
| ⏱️ Long-Distance Simulation | จำลอง signal propagation delay จริง |
| 🔧 Custom Protocol Design | ออกแบบ protocol เฉพาะสำหรับ interplanetary comms |

**Tech Stack:** `Python` `Socket API` `Threading` `Custom DTN Protocol`
